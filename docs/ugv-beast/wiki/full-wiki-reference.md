## Key Directory Paths on Robot

| Path | Purpose |
|------|---------|
| `/home/jetson/ugv_jetson/` | Main host program (Flask app) |
| `/home/jetson/ugv_ws/` | ROS2 workspace (host OS) |
| `/home/jetson/ugv_ws/ros2_humble.sh` | Docker entry script |
| `/home/ws/ugv_ws/` | ROS2 workspace (inside Docker) |
| `/home/ws/ugv_ws/src/ugv_main/` | Custom ROS2 packages |
| `/home/ws/ugv_ws/src/ugv_else/` | Vendored dependencies |
| `/home/ws/ugv_ws/src/ugv_main/ugv_nav/maps/` | Saved 2D maps |
| `/home/ws/ugv_ws/src/ugv_main/ugv_nav/launch/` | Nav launch files |
| `/home/ws/ugv_ws/map_points.txt` | Saved nav points |
| `~/.ros/rtabmap.db` | 3D map database |
| `/home/ws/ugv_ws/src/ugv_main/ugv_chat_ai/ugv_chat_ai/app.py` | AI chat app |
